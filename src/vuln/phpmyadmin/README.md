**简介：**[phpMyAdmin](https://www.phpmyadmin.net/) 是一个用PHP编写的免费软件工具，用于在Web上管理MySQL。phpMyAdmin支持MySQL和MariaDB的广泛操作。常用的操作(管理数据库、表、列、关系、索引、用户、权限等)可以通过用户界面执行，同时您仍然能够直接执行任何SQL语句。
