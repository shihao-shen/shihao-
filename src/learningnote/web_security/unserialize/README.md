## 什么是反序列化攻击？

首先简单说明一下序列化，序列化就是将对象转化为可以存储和传输的数据形式，反序列化就是将序列化后的数据转换为对象。

反序列化攻击，一般存在于面向对象的编程语言中，需要对源代码进行审计，找到可以进行利用的点，才可以构造POP链，实现对使用这套代码的目标进行攻击。不同的编程语言，反序列化攻击的方式也不一样，比如PHP和python的类对象是通过魔术方法进行利用，但是java却没有魔术方法，使用的是反射机制。
