### 漏洞描述

什么是信息泄露漏洞？信息泄露漏洞是指客户端在未获得授权的情况下，通过某种手段（目录扫描、查看网页源码等），访问并下载了网站内未公开的重要文件（配置文件、日志文件、备份文件等）
